# pdfs/

Source DIPRES *Informe de Pasivos Contingentes* (Chilean government contingent liabilities reports), one PDF per year, 2019–2025.

These are the working source documents from which contingent-liabilities data (TOTAL row of "Cuadro 2", as % of GDP) was extracted into `data/manual/contingent_liabilities.csv`. They are not read by the pipeline at runtime — extraction was a one-time, agent-assisted task.

Source: https://www.dipres.gob.cl (search for "Informe de Pasivos Contingentes").
